# Исследование объявлений о продаже квартир

## Постановка задачи

Анализ архива объявлений о продаже квартир в **Санкт-Петербурге и Ленинградской области** на основе данных сервиса Яндекс Недвижимость. Цель — определить параметры, влияющие на рыночную стоимость объектов, и подготовить основу для автоматизированной системы обнаружения аномалий и мошеннической деятельности.

## Данные

Файл `real_estate_data.csv` содержит объявления о продаже квартир. Каждый объект описан двумя типами данных: внесёнными пользователем и автоматически полученными из геосервисов.

| Столбец | Описание |
|---------|----------|
| `total_area` | Общая площадь (м²) |
| `living_area` | Жилая площадь (м²) |
| `kitchen_area` | Площадь кухни (м²) |
| `last_price` | Цена на момент снятия с публикации |
| `rooms` | Число комнат |
| `floor` / `floors_total` | Этаж / всего этажей |
| `ceiling_height` | Высота потолков (м) |
| `first_day_exposition` | Дата публикации объявления |
| `days_exposition` | Количество дней размещения |
| `locality_name` | Название населённого пункта |
| `cityCenters_nearest` | Расстояние до центра города (м) |
| `airports_nearest` | Расстояние до аэропорта (м) |
| `parks_nearest` | Расстояние до ближайшего парка (м) |
| `balcony` | Число балконов |

## Этапы работы

### 1. Предобработка данных
- Обработка пропусков: балконы → 0, потолки → медиана, площадь → медиана по числу комнат
- Исправление типов данных: `first_day_exposition` → datetime, `floor/rooms` → int
- Устранение неявных дубликатов в названиях населённых пунктов
- Обработка выбросов: высота потолков > 10 м делится на 10 (артефакты ввода)

### 2. Новые признаки
- `price_per_sqm` — цена за м²
- `weekday_exposition`, `month_exposition`, `year_exposition` — дата публикации
- `floor_type` — тип этажа: первый / последний / другой
- `city_center_km` — расстояние до центра в км (округлено)

### 3. Ключевые результаты

**Скорость продажи**: медиана — ~95 дней. Быстрые продажи — до 30 дней, долгие — свыше 6 месяцев.

**Факторы цены**:
- Сильная корреляция: общая и жилая площадь, количество комнат
- Квартиры на первом этаже стоят дешевле, на последнем — незначительно дешевле среднего
- Цена снижается по мере удаления от центра Санкт-Петербурга

**Населённые пункты**: Самые дорогие м² — в Санкт-Петербурге, самые доступные — в пригородах.
